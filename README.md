## Установка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm dev

# Сборка для production
pnpm build

# Генерация статического сайта
pnpm generate
```

## Игровая механика

- **Автономный герой** - персонаж действует самостоятельно
- **Боевая система** - автоматические бои с врагами
- **Система классов** - Воин и Маг со своими способностями
- **Прокачка** - уровни, опыт, характеристики
- **Инвентарь** - предметы, зелья, свитки
- **Торговля** - автоматическая покупка и продажа
- **Квесты** - задания на прохождение 500 ходов
- **Локации** - разные зоны с разным уровнем опасности

## Скрипты

- `pnpm dev` - Запуск dev сервера
- `pnpm build` - Сборка для production
- `pnpm generate` - Генерация статического сайта
- `pnpm preview` - Предпросмотр production сборки
